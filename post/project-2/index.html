INTRODUCTION

Amazon Review sentiment analysis with NLP. This is a Sentiment Analysis project using Natural Language Processing(NLP).The data contained the people’s revies on amazons products, labels(the sentiment of people which are in zero’s (0) and one’s (1) where zero means the person is angry or sad or basically something negative while the one means a positive review and the id(which of course is irrelevant to us). The aim of this project is to compare the Naïve Bays model and the Logistic regression model to test and train the dataset to find the unknown class(data) prediction.
